Программы центра — 3 концепции

Текущий блок неплохой, но цветная полоска не считывается, иерархия кнопок «Записаться / Подробнее» сбита, а возрастной чип — главный фильтр родителя — спрятан мелким. Три разных угла, как это починить.

V1 · Safe

Возраст во главе

Возрастной чип становится крупным заголовком карточки — родитель видит «5–7 лет» с первого взгляда. Цветная плашка вместо полоски, единственная CTA «Записаться», «Подробнее» — мелкая ссылка-стрелка. Чистая иерархия, всё в дизайн-системе.

Смотреть → ~1 ч фронта
V2 · Evolve

Список-сравнение + фильтр

Компактная таблица-список: направление, возраст, формат, цена, CTA. Слева — панель с фильтром по возрасту. Удобно сравнивать программы. Для главной — спорно, для каталога /programmy/ — почти готовое решение.

Смотреть → ~3 ч фронта
V3 · Bold

Цветные плитки-знакомства

Каждая программа — крупная цветная плитка из палитры цветка. Сверху иконка-объект, в центре возраст огромными цифрами, под ним одна выгода («заговорит без страха»), цена мелким, единая CTA. Эмоциональный ход в духе Lingokids — детям и родителям сразу тепло.

Смотреть → ~4 ч фронта
Как читать. Все три варианта рендерятся в реальной обвязке сайта (header, footer, шрифты, палитра). CSS блока — inline в шаблоне, чтобы ничего не утекло в прод. Прод-главная (/) не тронута. Данные программ — мок внутри view, БД не дёргаем.
