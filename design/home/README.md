# design/home — автономная главная

Снимок главной страницы сайта, собранный из живого рендера Flask.
CSS заинлайнен в `<style>`, картинки лежат в `img/`, скрипты в `main.js`.

Открыть локально: `open design/home/index.html` (двойной клик тоже работает).

## Для итераций в Клоде
Загружай `index.html` — это один самодостаточный файл (~170 КБ)
со всей вёрсткой и стилями главной. Можешь менять разметку/стили
прямо в нём, фронтендер потом переносит изменения в Jinja-шаблоны
и `app/static/css/main.css`.

## Что НЕ работает
- Клики по ссылкам вида `/programmy/`, `/o-centre/` — это роуты Flask,
  в standalone-файле они 404-ят. Для дизайн-итераций не нужно.
- Формы — без бекенда не отправляются. Только внешний вид.

## Пересобрать
Если потребуется обновить снимок (поменялся контент/блоки в БД):

```bash
FLASK_APP=wsgi.py python -m flask run --port 5001 &
curl -s http://127.0.0.1:5001/ -o design/home/_raw.html
# затем повторить inline CSS + rewrite /static/ → ./ (см. историю коммитов)
```
